# Share.dev

A nonsense Social Media clone.

Built using Golang + HTMX!

## Current Scope

- Decent Homepage
- Login / Signup pages
- Scalable User Sessions
- "Feed" Page
- Post Text

## Dev Notes:

Using Air for live reloading.  
Install air: `go install github.com/air-verse/air@latest`
